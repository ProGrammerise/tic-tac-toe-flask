# Tic Tac Toe Game 🎮

A responsive and interactive Tic Tac Toe web application built using Flask, HTML, CSS, JavaScript, and Bootstrap.

---

## 📌 Features

- Two-player Tic Tac Toe gameplay
- AI opponent mode
- Win and draw detection
- Restart game functionality
- Responsive UI design
- Clean and user-friendly interface
- Backend integration using Flask

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML5

- Git & GitHub

## 🚀 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ProGrammerise/tic-tac-toe-flask.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd tic-tac-toe-flask
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Flask Application

```bash
python app.py
```

### 5️⃣ Open in Browser

```plaintext
http://127.0.0.1:5000
```

## 🎯 Game Rules

- The game is played between two players.
- Players take turns marking X and O.
- The first player to align 3 symbols horizontally, vertically, or diagonally wins.
- If all cells are filled without a winner, the game ends in a draw.
